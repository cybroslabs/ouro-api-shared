Message SystemConfig

Messages

message SystemConfig

Field Type Description
1 max_replicas int32

The maximum number of replicas for the driver. 0 represents no active replicas will run, effectively disabling acquisition

0 represents the maximum number of replicas per driver
2 max_cascade_device_count int32

The maximum number of cascade devices for the driver. Minimum is 1.
3 max_slots_per_driver int32

The maximum number of slots per driver -1 represents unlimited number of slots, effecticaly using maximum number of slots supported by driver 0 represents no active slots will run, effectively disabling acquisition

0 represents the maximum number of slots per driver, the number of slots never exceeds the number of slots supported by driver
