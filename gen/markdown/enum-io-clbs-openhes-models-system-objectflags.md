# Enum: io.clbs.openhes.models.system.ObjectFlags

Defines feature flags and capability indicators for system objects.
These flags control what operations are available or enabled for specific objects.

## Options

| Value | Description |
| --- | --- |
| OBJECT_FLAG_UNSPECIFIED | Default/unspecified flag (no special behavior). |
| OBJECT_FLAG_COMMUNICATION_UNIT_LOG | The communication unit has detailed logging enabled for diagnostics and troubleshooting. |
