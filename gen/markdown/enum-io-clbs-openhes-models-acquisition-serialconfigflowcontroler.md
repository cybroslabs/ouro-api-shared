# Enum: io.clbs.openhes.models.acquisition.SerialConfigFLowControler

Defines the available serial port flow control types.

## Options

| Value | Description |
| --- | --- |
| FLOW_CONTROL_UNSPECIFIED | Unspecified flow control. |
| FLOW_CONTROL_NONE | No flow control. |
| FLOW_CONTROL_HARDWARE | Hardware flow control using RTS/CTS signals. |
| FLOW_CONTROL_SOFTWARE | Software flow control using XON/XOFF characters. |
